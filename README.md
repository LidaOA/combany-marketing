# Combany - Extension Marketing Site

This is the marketing landing page for the Combany browser extension.

## Overview
- It is fully static and ready to be hosted on Vercel or any other static hosting platform.
- It detects the user's IP address to automatically adapt the language (English/French).

## Deployment
This folder (`marketing`) is designed to be the "Root Directory" in your Vercel project settings. Simply import this directory and Vercel will serve `index.html`.
